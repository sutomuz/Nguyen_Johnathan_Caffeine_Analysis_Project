Data ReadMe.txt
Dataset Name: Caffeine Intake Tracker
File Included: caffeine_intake_tracker.csv
File Type: CSV

Description of the Dataset:
The dataset contains 500 observations are 13 variables:
Caffeine Intake (scaled and later converted to mg)
Age (0-1 scale)
Focus Level
Sleep Quality
Sleep Impact
Beverage Types (coffee, tea, energy drinks)
Time of Day caffeine was consumed (morning/afternoon/evening)
Gender (male/female)
The dataset is clean, tidy, and contains no missing values.
It was well structured for analysis and visualizations.
